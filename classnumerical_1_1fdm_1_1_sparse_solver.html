template<typename T>
class numerical::fdm::SparseSolver< T >

This class only computes the 1D diffusion problem with Dirichlet/Neumann boundary conditions and heterogenous diffusion coefficient (for now).

Member Function Documentation

template<typename T >
virtual void numerical::fdm::SparseSolver< T >::assemble_a ( )
inlinevirtual

Assembles the sparse coefficient matrix \(\mathbf{A}\). The matrix is defined by its diagonals. The number of diagonals is related to the number of direct neighbors for interior mesh nodes. In 1D, the matrix has 3 non-zero diagonals, in 2D it has 5, and in 3D, 7.

The diagonals of the matrix \(\mathbf{A}\) is are filled by vectorization of the loops for efficiency.

Vec &  u_n 
)
inlinevirtual

Assembles the RHS vector \(\mathbf{b}\).

\[ b_i = u_i^n + F\left(1-\Theta\right)u_{i+1}^n-2u_i^n+u_{i-1}^n + \Delta t \Theta f_i^{n+1} + \Delta t \left(1-\Theta\right)f_i^n \]

using vectorization we get:

\[ b[1:n_x-1] = u_n[1:n_x-1] + \left(1-\Theta\right)F \left(u_n[2:n_x]-2u_n[1:n_x-1]+u_n[0:n_x-2]\right) + \Theta\Delta t f[1:n_x-1](n+1) + \left(1-\Theta\right)\Delta t f[1:n_x-1](n) \]
