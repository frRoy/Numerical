Detailed Description

This class provides tools to compute the finite difference problem defined in FDMDiffusionD::reference().

The class is derived form the numerical::fdm::FDProblem class and overrides the member function numerical::fdm::FDProblem::left(), numerical::fdm::FDProblem::right(), numerical::fdm::FDProblem::initial_value(), numerical::fdm::FDProblem::source(), and numerical::fdm::FDProblem::reference().

Reference solution for the 1D diffusion problem d, defined as:

\[ \frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u} {\partial x^2} + f(x,t)\quad x \in (0, L),~t \in (0, T] \]

with non-homogeneous Dirichlet boundary condition \(u(0, t) = 5tL^2/4\) and \(\mathbf{\hat{n}}\cdot\left. \alpha(u)\frac{\partial}{\partial x} u(x, t)\right|_{x=L} = g\).

The diffusion coefficient, \( \alpha(u)=u\).

We define the source term and initial condition with a reference (manufactured) solution that respects the boundary conditions at the extremities of the interval:

\[ u(x, t) = 5t\left(x-\frac{L}{2}\right)^2 \]

We get the source term substituting the manufactured solution in the diffusion equation, i.e.:

\[ f(x, t) = 5\left(x-\frac{L}{2}\right)^2- 10\alpha t \]

The initial condition is simply set to the manufactured solution at \(t=0\):

\[ u(x, 0) = u_0 = 0 \]
