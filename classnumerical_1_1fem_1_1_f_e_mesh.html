Detailed Description

template<typename T>
class numerical::fem::FEMesh< T >

Holds data structures for a uniform mesh on a line in space, plus a uniform mesh in time.

The finite element mesh has n elements and an associated polynomial of degree d.

Parameters
lengthsA 2-lists of min and max coordinates in the spatial direction.
t0The initial time in time mesh.
tendThe final time in time mesh.
ntNumber of cells in time mesh.
nNumber of cells in the spatial direction.
dThe polynomial degree, default = 1.
