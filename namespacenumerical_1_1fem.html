template<typename T >
std::vector<std::vector<T> > numerical::fem::GaussLegendre ( int  n)

Return points and weights for Gauss-Legendre rules on [-1,1]. The number of points implemented are 1-4.

Numerical integration rules can be expressed in a common form:

\[ \int_{-1}^{1} g(X)dX\approx \sum_{j=0}^M w_jg(\bar{X}_j) \]

where \(\bar{X}_j\) are integration points, and \(w_j\) are integration weights, for \(j=0,\ldots ,M\).

More accurate rules, for a given \(M\), arise if the location of the integration points are optimized for polynomial integrands. The Gauss-Legendre quadrature constitute one such class of integration methods. Two widely applied Gauss-Legendre rules in this family have the choice:

for \(M=1\):

\[ \bar{X}_0 = -\frac{1}{\sqrt{3}},\quad \bar{X}_1 = \frac{1}{\sqrt{3}}, \quad w_0=w_1=1 \]

For \(M=2\):

\[ \bar{X}_0 = -\sqrt{\frac{3}{5}},\quad \bar{X}_1 = 0, \quad \bar{X}_2 = \sqrt{\frac{3}{5}}, \quad w_0=w_2=\frac{5}{9}, \qaud w_1 = \frac{8}{9} \]

These rules integrate 3rd and 5th degree polynomials exactly. In general, an \(M\)-point Gauss-Legendre rule integrates a polynomial of degree \(2M + 1\) exactly.

Returns the first derivative of the \(i\)-th Lagrange polynomial.

\[ \begin{align} \frac{\partial}{\partial x}\phi_i(x) &= \frac{\partial}{\partial X}\phi_i(X)\frac{\partial X}{\partial x}\\ &= \frac{\partial}{\partial X}\phi_i(X)\frac{2}{h} \end{align} \]

For details see Langtangen2019, p.188 Cellwise computation.

The first derivative of the Lagrange polynomial with respect to local coordinates \(X\) can be obtained using the logarithm of the Lagrange polynamial, i.e.

\[ \textrm{ln}\left(\prod_{j=0, j\neq i}^N \frac{x-x_j}{x_i-x_j}\right) = \sum_{j=0, j\neq i}^N \textrm{ln}\left(\frac{x-x_j}{x_i-x_j}\right) \]

The derivative of the logarithm is

\[ \frac{\partial}{\partial X}\left(\textrm{ln}(f(x))\right) = \frac{1}{f(X)} \frac{\partial}{\partial X}f(X) \]

with \(f(x)=\frac{x-x_j}{x_i-x_j}\), we get

\[ \frac{\partial}{\partial X}\phi_i(X) = \phi_i(X)\sum_{j=0, j\neq i}^N \frac{1}{x-x_j} \]

Note that this result is only valid for \((x-x_j)\neq 0\).

Returns all local basis function \(\phi\) and their derivatives, in physical coordinates, as functions of the local point \(X\) in a reference cell with \(d+1\) nodes.

Example:

#include "numerical/fem/FiniteElement.hpp"
using namespace numerical;
int main()
{
double f0 = fem::phi(0, 0, 0.0); // basis func 0 at X=0
double fp0 = fem::phi(0, 0, 0.0); // 1st x-derivative at X=0
spdlog::info("f0: {}, fp0: {}", f0, fp0);
return 0;
}

Output:

1 $ f0: 0.5, fp0: -2.0
