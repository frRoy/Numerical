Detailed Description

template<typename T>
class numerical::fem::FEProblem< T >

Defines the finite element problem over a line.

This class only define the 1D diffusion problem with Dirichlet/Neumann boundary conditions and heterogenous diffusion coefficient (for now).

Support for hyperbolic equations (wave and convection diffusion) will be added later.
